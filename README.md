# Equinoxio Mapping Suite — Descargas

Herramientas de ingeniería para proyección profesional: videomapping, salas inmersivas y sistemas multiproyector. **Calcula la instalación antes del montaje**: resolución, blending, lentes, brillo, servidor y pixel maps.

> 100% local · sin nube · sin suscripción.

## Descargar
Los instaladores están en **[Releases →](../../releases/latest)**

- **Windows (x64):** `Equinoxio Mapping Suite Setup 0.9.0.exe` — SmartScreen mostrará un aviso (app sin firma): *Más información → Ejecutar de todas formas*.
- **macOS (Apple Silicon):** `Equinoxio Mapping Suite 0.9.0 arm64.dmg` — primera apertura: *clic derecho → Abrir*.
